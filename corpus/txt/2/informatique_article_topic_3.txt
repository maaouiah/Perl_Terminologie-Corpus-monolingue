                                                                                                                &nbsp Installation dune machine virtuelle VMware   
                                      
                                                                                      VMware Server permet de simuler un PC sur votre PC. Cela permet par exemple de faire tourner Linux sur votre Windows XP sans jouer avec les partitions de windows mais en utilisant un disque virtuel. VMware permet de faire fonctionner plusieurs systèmes d'exploitation en même temps sur une seule et même machine physique grâce à la technologie de virtualisation.Pour faire simple si vous virtualisez Linux Debian grâce à une machine virtuelle sur Windows XP, vous aurez sur votre bureau une banale fenêtre d'application, mais à l'intérieure de cette fenêtre, il y aura une Linux Debian comme si vous l'aviez installée normalement sur votre ordinateur.Il y a de nombreux avantages a utiliser des machines virtuelles :- Pas besoin de redémarrer votre ordinateur pour changer d'OS- Vous ne risquez pas de supprimer les données de vos partitions puisque tout se passe sur un disque virtuel- Vous pouvez tester facilement et rapidement différents OS
